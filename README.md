# windblog

#### 介绍
前后端分离的个人博客网站

#### 软件架构
windblog-web 前端
windblog-api 后端

#### 安装教程
1. git clone https://gitee.com/zhixingqingnian/windblog.git
2. cd windblog-web
3. pnpm i

#### 使用说明
1. 网页分页前台展示合后台管理
2. 可使用 test/123456,体验页面功能
